# Demo

Deze map bevat demo-applicaties.

- `m2m/web`: actieve M2M web demo die door Tilt gebouwd wordt.
- `m2m/console`: console demo voor de client credentials flow.
- `m2m/seed`: seed console die vaste testdata voorziet voor de M2M console demo.
- `nuxt-bff`: Nuxt BFF demo die door Tilt gebouwd wordt.

Lokale Kubernetes- en Helm-configuratie staat onder `local-dev`.
