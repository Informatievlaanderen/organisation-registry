Voer de volgende stappen uit:

1. Voer `git status` en `git diff --cached` uit om te zien wat er gecommit wordt.
2. Maak een commit met een conventional commit message (fix:/feat:/docs:/refactor: etc.), gebaseerd op de wijzigingen. Geen Co-Authored-By.

Argumenten (optioneel): $ARGUMENTS — als meegegeven, gebruik dit als commit message of hint.
