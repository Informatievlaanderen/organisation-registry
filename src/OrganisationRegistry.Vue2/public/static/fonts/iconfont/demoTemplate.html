第一步：使用font-face声明字体 
            @font-face {
                font-family: "vlaanderen_iconfont";
                src: url('../fonts/iconfont/vlaanderen_iconfont.eot');
                src: url('../fonts/iconfont/vlaanderen_iconfont.eot?#iefix') format('eot'),
                    url('../fonts/iconfont/vlaanderen_iconfont.woff') format('woff'),
                    url('../fonts/iconfont/vlaanderen_iconfont.ttf') format('truetype'),
                    url('../fonts/iconfont/vlaanderen_iconfont.svg#vlaanderen_iconfont') format('svg');
            }
第二步：定义使用iconfont的样式 
.vi{
    font-family:"vlaanderen_iconfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取相应class，应用于页面,【vi-xxxx】=【vi-】前缀+【每个图标对应的class】 
<i class="vi gm-xxxx"></i>
